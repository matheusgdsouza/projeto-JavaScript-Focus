# 📌 Fokus - Aplicação de Produtividade (Pomodoro)

## 🧠 Sobre o projeto

O **Fokus** é uma aplicação web desenvolvida com o objetivo de auxiliar na produtividade utilizando a técnica **Pomodoro**, permitindo ao usuário gerenciar seu tempo de foco e suas tarefas de forma simples e eficiente.

O projeto foi desenvolvido com foco na prática de conceitos essenciais de **JavaScript moderno**, principalmente relacionados à manipulação do DOM, controle de tempo e persistência de dados no navegador.

---

## 🚀 Funcionalidades

* ⏱️ Temporizador no estilo Pomodoro (Foco, descanso curto e longo)
* ▶️ Controle de execução (iniciar, pausar e resetar)
* 📝 Gerenciamento de tarefas (adicionar, editar e remover)
* 💾 Persistência de dados utilizando **LocalStorage**
* 🎨 Alteração dinâmica de elementos visuais conforme o estado do app
* 🔊 Feedback sonoro para melhorar a experiência do usuário

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)

---

## 📚 Principais conceitos de JavaScript aplicados

### 🔹 Manipulação do DOM

* Seleção de elementos (`querySelector`, `getElementById`)
* Alteração dinâmica de conteúdo (`innerText`, `innerHTML`)
* Manipulação de classes (`classList.add/remove/toggle`)
* Alteração de atributos e estilos

👉 Essencial para criar interfaces dinâmicas e reativas no navegador.

---

### 🔹 Eventos e Interações

* `addEventListener`
* Eventos de clique e interação do usuário
* Controle de estado da aplicação baseado em ações

👉 Permite transformar uma página estática em uma aplicação interativa.

---

### 🔹 Controle de Tempo

* Uso de `setInterval` e `clearInterval`
* Lógica de contagem regressiva
* Alternância entre modos (foco, descanso curto e longo)

👉 Fundamental para aplicações que envolvem tempo real.

---

### 🔹 Estruturas de Dados

* Arrays para gerenciamento de tarefas
* Objetos para modelagem de dados
* Métodos como `map`, `filter` e `forEach`

👉 Base para organização e manipulação de dados no frontend.

---

### 🔹 LocalStorage (Persistência de dados)

* Armazenamento de tarefas no navegador
* Conversão de dados com `JSON.stringify` e `JSON.parse`

👉 Permite manter os dados mesmo após recarregar a página.

---

### 🔹 Funções e Organização de Código

* Funções reutilizáveis
* Separação de responsabilidades
* Código mais limpo e escalável

---

### 🔹 Lógica de Programação

* Condicionais (`if/else`)
* Controle de fluxo
* Estados da aplicação

👉 Essencial para resolver problemas e construir aplicações reais.
